# Positivus – Digital Marketing Agency

[Ссылка на GitHub Pages](https://nadezhdamak.github.io/Positivus/)

## О проекте
Адаптивный лендинг для digital-маркетингового агентства.
Проект свёрстан по БЭМ, адаптирован под разные устройства, с учетом семантики и доступности для скринридеров.

## Технологии

- HTML
- SCSS
- БЭМ
- JavaScript (ES6)
- Адаптивная верстка

## Адаптивность

Поддерживаются следующие ширины:

- Десктоп: от `1240px`
- Планшет: `1040px`
- Мобильные устройства: `768px`
- Дополнительно: узкие экраны от `340px` (тестировалось на Samsung Galaxy Z Fold 5)
